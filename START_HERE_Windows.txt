TEST REPORT ANALYZER - Windows quick start
==========================================

ONE-TIME SETUP
--------------
1. Install Python 3 from  https://www.python.org/downloads/
   IMPORTANT: on the first installer screen, tick the box
   "Add python.exe to PATH", then click Install.

HOW TO USE IT
-------------
Easiest - drag and drop:
   Drag a FOLDER of PDF reports (or some selected PDF files)
   onto "run.bat".
   - The first run installs what it needs (needs internet, ~1 minute).
   - When it finishes, an "output" folder opens containing:
        TR_Summary.xlsx   <- your FX / SD summary, one row per specimen
        summary.csv

Or use the reports folder:
   1. Double-click "run.bat" once. It creates a "reports" folder.
   2. Put your PDF reports into that "reports" folder.
   3. Double-click "run.bat" again.

Results are rewritten into the "output" folder every time you run it.

OPTIONAL - append into your OWN template instead of a new file
--------------------------------------------------------------
Open a Command Prompt in this folder (type "cmd" in the address bar of
the File Explorer window, then Enter) and run:

   py -m report_analyzer reports --xlsx output\filled.xlsx --xlsx-template "YourTemplate.xlsx"

This keeps your template's existing rows and styling and adds new rows
below them (windows on the FX sheet, doors on the SD sheet).

TROUBLESHOOTING
---------------
- "Python is not installed" -> install it (step 1) and tick "Add to PATH".
- A black window flashes and closes -> right-click run.bat > Edit to read
  messages, or run it from a Command Prompt to see the output.
