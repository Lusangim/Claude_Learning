TEST REPORT ANALYZER - quick start
==================================

ONE-TIME SETUP
--------------
1. Install Python 3 from  https://www.python.org/downloads/
   IMPORTANT: on the first installer screen, tick the box
   "Add python.exe to PATH", then click Install.

HOW TO USE IT
-------------
1. Put your PDF reports into the "reports" folder.
2. Run the launcher, either way:
      - Double-click "run.py"  (works if Python is installed), OR
      - Open a terminal / Command Prompt in this folder and type:
            python run.py
   The first run installs what it needs (needs internet, ~1 minute).
3. When it finishes, the "output" folder contains:
      TR_Summary.xlsx   <- your FX / SD summary, one row per specimen
      summary.csv

To analyze a different folder instead of "reports":
      python run.py "C:\path\to\your\reports"

HANDLING ANY REPORT FORMAT (AI MODE)
------------------------------------
The built-in parser knows two common lab formats. For other companies/formats,
use AI mode, which reads any report with Claude. You need an API key from
https://console.anthropic.com (small cost per report). In a Command Prompt here:

   set ANTHROPIC_API_KEY=sk-ant-your-key-here
   python run.py

(run.py uses AI automatically once the key is set. There is also
 "python -m report_analyzer reports --mode ai --xlsx output\TR_Summary.xlsx".)

OPTIONAL - append into your OWN template instead of a new file
--------------------------------------------------------------
      python -m report_analyzer reports --xlsx output\filled.xlsx --xlsx-template "YourTemplate.xlsx"

TIPS
----
- To open a Command Prompt in this folder: type "cmd" in the address bar of
  the File Explorer window showing these files, then press Enter.
- "Python is not installed / not recognised" -> install it (step 1) and make
  sure you ticked "Add python.exe to PATH".
